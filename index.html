Entornos Virtuales

Opcion 1: VirtualEnv

  1. Instalación (LINUX) 
    :~$ sudo apt-get install virtualenv python-virtualenv
:~$ sudo pip install virtualenv
  2. Ejecutar: Se crea carpeta "mi entorno" y subcarpetas "bin, include, lib"
    3. :~$ virtualenv nombreentorno
  3. Activar: Entro a la carpeta "entornovirtual" 
    :~$ source bin/activate
(nombreentorno):~$
  4. Instalar paquetes:
    5. (nombreentorno):~$ pip install numpy
  5. Desactivar:
    6. (nombreentorno):~$ deactivate

OPCION 2

  • Conda (Anaconda)
:~$ conda create -n mientornovirtual python=x.x
:~$ source activate mientornovirtual
:~$ source deactivate
:~$ conda info --envs

Recomendación: Link a Conda vs. pip vs. virtualenv
